## 📄 Project Overview
This project focuses on executing advanced statistical plotting across multiple data science environments. The analysis utilizes the **Womens Clothing E-Commerce Reviews** dataset to uncover hidden distribution geometries, core departmental trends, consumer rating densities, and structural outperforming segments.

The project is structured into three distinct execution steps:
1. **Theoretical Research:** Comprehensive documentation of charting properties, including core trade-offs, advantages, and use cases.
2. **Python Exploratory Framework:** Scripted analytical visualizations leveraging high-level graphing architectures.
3. **KNIME Analytical Pipeline:** Building zero-code processing and reporting pipelines to validate data consistency.

---

## 📊 Core Charting Implementations & Statistical Scope

### 1. Box Plot (Kutu Grafiği)
* **Objective:** Map structural data distributions, dispersion scales, and pinpoint explicit dataset outliers.
* **Domain Context:** Evaluating interquartile ranges (IQR) of consumer ratings across corporate retail branches.

### 2. Violin Plot (Keman Grafiği)
* **Objective:** Model probability densities using Kernel Density Estimation (KDE) coupled with baseline median summaries.
* **Domain Context:** Profiling consumer rating distributions and densities across precise retail departments.

### 3. Line Graph (Çizgi Grafik)
* **Objective:** Track and continuous value trend variations across sequential indices.
* **Domain Context:** Mapping consumer feedback behaviors and review sequences.

### 4. Bar Graph (Sütun Grafiği)
* **Objective:** Execute clear frequency and volumetric value comparisons across distinct categorical segments.
* **Domain Context:** Summing absolute review volumes and categorical interaction magnitudes.

### 5. Pie Chart (Pasta Grafiği)
* **Objective:** Dissect proportional market compositions and categorical part-to-whole segment representations.
* **Domain Context:** Illustrating macro departmental share distributions within comprehensive e-commerce revenue models.
